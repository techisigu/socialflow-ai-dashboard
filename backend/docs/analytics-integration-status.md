# Analytics Integration Status

This document tracks the current implementation status of analytics integrations by platform and capability.

## Status Legend

- **implemented**: Production code path exists and is wired for the capability.
- **partial**: Some building blocks exist, but the capability is not fully wired end-to-end.
- **planned**: Stub/TODO only; no active implementation.
